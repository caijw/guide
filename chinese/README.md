# Deno Core指南

## 这是什么？

本指南介绍了Deno（一个安全的服务端的TypeScript运行时）的设计和架构。 

本指南是由Deno的贡献者创建和维护的。

## 在你开始之前

如果你对Deno还不是很熟悉，我们建议你先查看下面的资源：

1. [10 Things I Regret About Node.js - Ryan Dahl - JSConf EU 2018](https://www.youtube.com/watch?v=M3BM9TB-8yA)
